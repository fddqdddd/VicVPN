# VicVPN — руководство пользователя

## Установка

1. Скачайте `VicVPN-0.1.0-setup-win64.exe` или portable ZIP.
2. Установщик требует права администратора (для TUN).
3. Portable: распакуйте ZIP в любую папку, запускайте `VicVPN.exe`.

## Первый запуск

1. Запустите **VicVPN от имени администратора** (ПКМ → Запуск от имени администратора).
2. Нажмите **+** и добавьте ключ или подписку.
3. Выберите сервер в списке → **Подключить**.

## Импорт конфигураций

| Формат | Пример |
|--------|--------|
| VLESS, VMess, SS, Trojan | `vless://...`, `vmess://...` |
| Hysteria2 | `hy2://...` |
| Подписка | `https://...` |
| ssconf | `ssconf://...` (выберите страну при импорте) |

Для **ssconf**: при импорте выберите страну (США, Нидерланды и др.) или «Авто».  
Сменить страну позже: правый клик по серверу → «Сменить страну».

## Подключение

- **Ваш IP** — реальный адрес без VPN.
- **IP VPN** — адрес через туннель (должен отличаться).
- **Сессия** — время текущего подключения.

TUN (весь трафик) работает только с правами администратора.

## Настройки

- **Обход локальной сети** — LAN без VPN.
- **Удалённый DNS** — DNS через туннель.
- **Kill switch** — блок при обрыве VPN.
- **Автозапуск** — старт с Windows.
- **Сворачивать в трей** — закрытие окна сворачивает в трей.

## Обновление подписок

Кнопка **Обновить подписки** перезагружает конфиг с сервера (HTTP и ssconf).

## Ядра

В папке `core\` должны быть: `xray.exe`, `hysteria.exe`, `wintun.dll`, geo-файлы.  
В установленной/portable-версии они уже включены.
